## 规则读取顺序
1. 先读 `全量开发规范法则/README.md`
2. 再读 `全量开发规范法则/00-开工门禁法则.md`
3. 再读 `全量开发规范法则/01-全量开发规范法则总典.md`
4. 再按任务主题读 `全量开发规范法则/02-主法分卷/`
5. 最后命中 `全量开发规范法则/03-细颗粒专题法则/` 中对应文件

## 当前口径
1. `全量开发规范法则/` 是唯一现行规范入口。
2. `Ai开发规则规范/` 已退出现行链路，只保留归档输入地位。
3. AI 与人工开发者都必须按“门禁法则 -> 总典 -> 主法分卷 -> 专题法则”的链路取法。
4. 任何任务都必须能在 3 跳内定位到执行依据。
